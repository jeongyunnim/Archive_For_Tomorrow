# CSRF, XSS, SF
### CSRF
- 이미지 src를 악의적으로 조작해서 GET 요청을 렌더링 시 보내도록 하는 공격
### XSS
- 스크립트 태그를 악의적으로 조작하여 실행하도록 하는 공격
### SF
- 세션 ID가 고정되어있을 경우 해커가 사용자에게 세션 ID를 넣은 페이지를 전달한다.
	- 전달된 페이지에서 로그인 할 경우 세션 아이디가 로그인이
