# 자바의 자료형
## 정수형
- byte
	- 래퍼 클래스: **Byte**
	- Byte.SIZE == 1;
	- Byte.BYTES == 8;
	- Byte.MAX_VALUE == 127;
- short
	- 래퍼클래스: **Short**
	- 2byte
- int 
	- 래퍼클래스: **Integer**
	- 4byte
- long
	- 래퍼클래스: **Long**
	- 8byte
	- 초기화 시 타입을 명시해주어야 한다.
		- `long longNum = 1000000000l;`
### 형변환
```java
int i;
long l = 10000000000l;

// i = l; compile error
i = (int)l;
```
