# 가변적 매개변수
```java
jshell> void print(int... values) {
   ...>     System.out.println(Arrays.toString(values));
   ...> }
|  created method print(int...)

jshell> print(1, 2, 3, 4, 5, 6, 8);
[1, 2, 3, 4, 5, 6, 8]
```
- 매개변수로 받은 인자들은 배열이 된다.
- 가변인자는 마지막에 와야 한다.
