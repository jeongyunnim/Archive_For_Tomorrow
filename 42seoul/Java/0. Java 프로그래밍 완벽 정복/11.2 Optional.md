# Optional
- 자바8 부터 지원한다.
- null pointer exception을 피하기 위한 수단이다.
- 기존 방식
	- 임의의 값에 접근하려고 할 때, 결과로 아무것도 표시되지 않는다면 null이 반환된 것이다.
	- API의 사용자들은 null의 반환을 알 수 없다.
### Optional 방식
- Optional 객체를 통해 값이 있는지 없는지 검증부터 할 수 있다.
