# View 환경 설정
- [스프링 공식 튜토리얼](https://spring.io/guides/)
- 스프링 부트 thymeleaf viewName 매핑
	- `resources/templates/` + `{ViewName}` + `.html`
- static file은 `resources/static`에 넣어두면 된다.

> [!tip]
> devtools를 설치한 뒤 바꿀 파일을 재 컴파일(build 탭) 하면 재 렌더링 해준다.
