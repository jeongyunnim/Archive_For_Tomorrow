# 글로벌 필터
- 모든 라우팅에 대해 적용되는 필터
- 필터만 구현하면 특별한 설정 없이 적용된다.
- 클라이언트 요청의 흐름
	- 필터 - 마이크로서비스 - 필터
	- 같은 필터라도 마이크로서비스에 접근하기 이전에는 pre, 이후에는 post
	- 정수값 order를 가질dis 수 있다.
