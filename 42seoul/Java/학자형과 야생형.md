# 학자형과 야생형

개발 공부는 야생형 스타일로 학습하는 개발자가 빨리 성장한다.
내가 해결해야 할 문제가 무엇인지 안 뒤에 문제 해결에 필요한 기술들을 학습하기 때문이다.

---
### 생각(파생된 질문/생각)
