# 좋은 객체 지향 프로그래밍이란?
## 객체 지향 프로그래밍
### 특징
- 추상화
- 캡슐화
- 상속
- 다형성
### 다형성
- 유연하다, 변경에 용이하다.
	객체들이 서로 메시지를 통해 상호작용하도록 한다.
	컴포넌트를 쉽고 유연하게 변경하면서 개발할 수 있는 방법.
	핵심은 **다형성**이다.
