# JavaScript 기본 문법
- 변수 선언
	- `let`
- Variable Naming
	- camelCase
	- 글자 또는 숫자로 이루어져 있어야 함.
		- 숫자로 시작하는 변수 이름 제한
	- `$`로 시작하는 변수 이름 허용.
	- `_`로 시작하는 변수 이름 허용
- 이항 연산자 지원
	- `+, -, *, %, /`
	- `**`: 거듭 제곱 연산자.
		- `2 ** 3` == 8
### Data Types
- Numbers
- String
```js
let someVariable = 200;
let somthingDescription = `someVariable is ${someVariable}`;
// someVariable is 200
```
