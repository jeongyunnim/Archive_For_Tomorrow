# Object Oriented Programming
- 코드 내에서 현실의 개체로 작업을 하는 것.
## Class
### 클래스 선언
```js
class Product {
	title = "Default";
	imageURL;
	description;
	price;

	constructor(title, image, desc, price) {
		this.title = title;
		this.imageURL = image;
		this.description = desc;
		this.price = price;
	}
}
```
- 해당 방식으로 생성할 객체의 데이터를 미리 작성해줄 수 있다.
	- 일종의 blueprint
- 클래스의 선언 순서는 상관 없다.
	- js가 미리 분석을 마칠 것이다.
	- 클래스를 인스턴스화 하여 메서드를 사용한다면, 해당 함수는 클래스 선언 뒤에 와야 한다.
