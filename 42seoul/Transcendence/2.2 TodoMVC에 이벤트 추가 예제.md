# TodoMVC에 이벤트 추가 예제
### 관리해야 할 이벤트
- 항목 삭제
	- 행의 오른쪽 십자가 클릭
- 항목의 완료여부 토글
	- 행의 왼쪽의 원 클릭
- 필터 변경
	- 하단의 필터 이름 클릭
- 항목 생성
	- 상단 입력 텍스트에 값 입력, 키보드의 Enter 입력
- 완성된 모든 항목 삭제
	- 'Clear completed' 레이블 클릭
- 모든 항목의 완료 여부 토글
	- 왼쪽 상단 모서리의 V를 클릭
- 항목 편집
	- 행을 더블 클릭하고 값 변경, 키보드의 Enter 입력
