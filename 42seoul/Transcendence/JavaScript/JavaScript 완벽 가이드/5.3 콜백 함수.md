# 콜백 함수
- 무언가에 의해 불러와진다.
```js
someBtn.addEventListener('click', () => {
	console.log('clicked!');
});
```
- 브라우저는 이벤트 등록할 때 함수의 인자로 함수의 포인터를 가져갈 뿐이다.
- 이벤트 리스너
- 예측 불가능하거나 오래걸리는 경우 콜백함수를 작성하게 될 것이다.
