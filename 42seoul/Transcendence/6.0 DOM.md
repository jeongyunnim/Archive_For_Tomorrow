# DOM
: Document object model
- 브라우저는 JS와 소통하여 렌더링된 HTML 요소와 상호하게 해준다.
- 이것을 **DOM**이라고 한다.
- 로드 및 렌더링된 HTML 소스코드이다.
- DOM은 브라우저에 국한되지 않는다.
	- HTML 요소를 읽고 쓸 수 있는 다른 도구들이 많다.
	- Node.js에는 DOM 기능을 지원하지 않는다.
