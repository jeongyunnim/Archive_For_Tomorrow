# Primitive vs Reference Types
### 원시 값
- 종류
```
Strings, Numbers, Booleans, null, undefined, Symbol
```
- 일반적으로 Stack에 저장된다.
	- 실행시간이 긴 경우 힙에 저장될 수도 있음.
- 스택은 **실행 context를 관리하기 위해** 도입되었다.
	- 전역 변수는 anonymous의 context에 포함된다.
	- 각 지역 변수는 해당 함수의 context에 포함된다.
### 참조 값
- 모든 object에 해당한다.
