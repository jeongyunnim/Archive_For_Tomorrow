# 3회 UART

- 시리얼 통신과 병렬 통신
	- 병렬통신은 거리가 멀 수록 와이어의 길이를 맞춰야 한다.
		- 구부러지는 경우 저항이 생길 수 있다.
- 동기 통신과 비동기 통신
### 동기 통신과 비동기 통신
- 비동기 통신
	- 별도의 클럭 신호 없이 
	- start bit와 stop bit 가지고 통신
	- 외부로 선이 나와있는 경우 비동기 통신을 사용한다.
