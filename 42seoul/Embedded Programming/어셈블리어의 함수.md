# 어셈블리어의 함수

- 함수 반환 시 PC를 POP하는 이유
```
push {r7, lr}

pop {r7, pc}
```
- lr 값을 push를 해놓고 pc로 pop을 해서 나온다.
- push, pop 개수를 맞춰줘야 한다.
- Interrupt 핸들러를 수행할 시, context(stack frame) 정보 8개를 푸시해놓았기 때문에 0xfffffff9를 읽고 MSP에서 정보를 context를 복원한다.
