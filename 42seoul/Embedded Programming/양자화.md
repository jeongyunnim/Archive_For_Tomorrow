# 양자화 量子化, quantization

입력 신호를 유한한 개수의 값으로 근사화하는 것. 비선형 연산이며, 원신호의 완전한 재생이 불가능하다. 입력 신호에 따라 스칼라 양자화와 벡터 양자화로 나눌 수 있으며, 부호어의 배치에 따라 균일 양자화와 비균일 양자화로 나눌 수 있다. 양자화된 결과는 디지털값으로 전송된다
