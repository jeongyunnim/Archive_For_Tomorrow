# 객체란 무엇인가?
- 객체는 실제 개체를 나타낼 때 사용할 수 있다.
	- 현실 세계 논리를 코딩에 적용할 수 있게 된다.
- JavaScript의 객체는 데이터 구조이다.
	- 키:값 쌍의 프로퍼티 혹은 메서드로 이루어져 있음.
	- 객체를 통해 데이터를 그룹화할 수 있다.
- 두 가지 유형의 값
	- 원시 값
	- 참조 값(객체)
- 객체 리터럴 표기 방식
```js
const person = {
	name: 'Max',
	age: 30,
	hobbies: ['sports', 'cooking'],
	greet: function() {
		alert('Hi there!');
	}
};
```
