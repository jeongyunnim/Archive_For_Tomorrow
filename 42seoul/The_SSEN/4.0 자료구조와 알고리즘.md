## 자료구조
### ADT
: Abstract Data Type
- 자료구조의 동작방식을 표현하는 데이터 형식
- 자료구조가 갖춰야할 일련의 연산
- 자료구조의 효용
	- 자료구조의 내부를 이해하면 라이브러리에서 엉뚱한 자료구조를 선택하는 일을 피할 수 있다.

- pool
