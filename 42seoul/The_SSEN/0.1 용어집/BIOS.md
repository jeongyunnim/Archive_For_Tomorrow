# BIOS
: Basic Input/Output System
- 운영 체제 중 가장 기본적인 소프트웨어이자 컴퓨터의 입출력을 처리하는 [[펌웨어]]
- 사용자가 컴퓨터를 켜면 시작되는 프로그램.
- 주변장치([[peripheral]])와 컴퓨터 운영체제 사이의 데이터 흐름을 관리한다.
