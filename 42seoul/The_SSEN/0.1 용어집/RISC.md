# RISC
: Reduced Instruction Set Computer
#### **RISC**의 특징
- 저전력 프로세서를 만들고자 하는 움직임으로 RISC 설계
- 제어유닛과 디코더를 매우 효율적으로 개선했다.
- 명령어를 모두 4바이트로 통일 파이프라인에 굉장히 유리
- 리스크 성능이 떨어진다.
- 레지스트가 많이 필요하다.
